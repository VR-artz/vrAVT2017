Instructions

  • Select your boad and change the fill color to your favorite color
  • Create a triangle that is 604 x 540, and set the x axis to 658, and the y axis to 540
  • Change the fill color of the triangle to the color of the shirt you are currently wearing.
  • Create another triangle that is 780 x 385, and set the x axis to 570, and the y axis to 182
  • Change the fill color of the triangle to the color of your hair
  • Create a circle that is 549 x 489, and set the x axis to 686, and the y axis to 171
  • Change the fill of the circle to a color as close to your own skintone as possible
  • Make another circle that is 89 x 84, and set the x axis to 809, and the y axis to 374
  • Change the fill of the circle to a color close to your eye color
  • Duplicate the circle and set the x asis to 1025 and the y axis to 374
  • Make another triangle that is 127 x 91, and set the x axis to 894.71, and the y axis to 49.74
  • Set triangles rotation to -177 degrees
